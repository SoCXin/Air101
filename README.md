## [描述](https://github.com/sochub/STM32F3/wiki) 

STM32F3系列与[STM32F1](https://github.com/sochub/STM32F1)系列引脚兼容，提供了cortex M3向cortex M4内核迁移途径，相比M4内核的[STM32F4](https://github.com/sochub/STM32F4)系列有更低的价格。
